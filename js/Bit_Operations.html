# JavaScript基础 - 位运算

# 基础

ECMAScipt中所有数值都以IEEE 754 64位格式存储，但是，位运算是先将值转换为32位整数，在进行位操作，之后再将结果转回64位。

# 运算基础

异或运算：

x ^ 0 = x​
x ^ 1 = ~x

与运算：

x & 0 = 0
x & 1 = x

# 位操作符

左移: 左移保留符号，右侧填0

1 << 1; // 2, 10
-1 << 1; // -2,

无符号右移： 将负数作为无符号数字处理，左侧填0右移

2 >>> 1; // 1
-1 >>> 1; // 2**31 - 1
